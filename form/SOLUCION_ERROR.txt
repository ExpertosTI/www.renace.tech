# GUÍA DE SOLUCIÓN DE ERRORES (ERROR JSON / PERMISOS)

Si ve el error `Unexpected token '<'`, significa que PHP está gritando porque no tiene permisos.

## SOLUCIÓN DEFINITIVA (Hacer esto UNA vez):

En su terminal del servidor (la misma donde tiene Docker), ejecute esto para "desbloquear" el archivo:

```bash
chmod 777 /var/www/renace.tech/form/data.json
```

**¿Por qué?**
Docker está montando el archivo de su servidor real. Si su servidor real dice "SOLO YO (Root) PUEDO ESCRIBIR", entonces el pobre PHP dentro de Docker (que es usuario `www-data`) rebota y falla.

Al darle permisos **777**, le dice al archivo: "Deja que cualquiera escriba en ti". Y así PHP puede guardar los datos.
